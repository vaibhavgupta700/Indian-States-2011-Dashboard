# Indian-States-2011-Dashboard
1️⃣ Project Description

Indian States 2011 Dashboard

This project is an interactive data visualization dashboard built using Microsoft Power BI to analyze demographic and geographic information of states in India based on Census of India 2011 data.

The dashboard provides insights into population distribution, capital cities, official languages, and state formation details. It helps users explore state-level information visually through maps, charts, and KPI cards.

2️⃣ Purpose of the Project

The purpose of this dashboard is to:

Visualize state-level demographic information

Provide interactive exploration of Indian states

Help users understand population distribution across states

Present key state information like capital, language, and formation date

This project demonstrates data analysis, visualization, and storytelling skills for a Data Analyst role.

3️⃣ Technology Used

Tools and technologies used in the project:

Microsoft Power BI – Data visualization & dashboard creation

Microsoft Excel – Data cleaning and preprocessing

Power Query – Data transformation

DAX (Data Analysis Expressions) – Creating calculated fields and metrics

4️⃣ Business Problem

Organizations and researchers often face difficulty in:

Understanding population distribution across states

Accessing state-level information in one place

Comparing demographic statistics across regions

Without visualization, the data from Census of India 2011 is difficult to interpret.

This dashboard solves that problem by providing clear visual insights and interactive exploration.

5️⃣ Dashboard Goal

The main goal of the dashboard is to:

Provide quick insights into Indian state demographics

Allow users to compare states easily

Display important state-level information visually

Enable data-driven understanding of population patterns

6️⃣ Charts Used in the Dashboard
1️⃣ Map Chart

Purpose:
Shows geographical location of states in India

Description:

Displays state markers on the India map

Allows users to explore geographic distribution

Business Value:
Helps understand regional distribution of data

2️⃣ Bar Chart – Population by State

Purpose:
Compare population between states.

Description:

X-axis → States

Y-axis → Population

Business Value:
Quickly identifies high and low population states.

3️⃣ KPI Cards

Purpose:
Display key information.

Examples visible in your dashboard:

Total Population

State Formation Date

Capital

Official Language

Business Value:
Shows important information instantly without analyzing charts.

4️⃣ Filters / Slicers

Purpose:
Allows users to select a specific state.

Description:
When a state is selected, the dashboard updates automatically.

Business Value:
Improves interactive analysis.

Different states have different official languages

State formation years vary widely

Population distribution is uneven across regions
